# Responsive Book Shop Website

This project is a frontend web development project built using HTML and CSS.  
The goal was to create a responsive web shop interface with book categories, product cards, and a static shopping cart.

## Features
- Responsive page layout using **CSS Grid and Flexbox**
- Semantic HTML structure
- Sidebar with book categories
- Grid display of books
- Book cards with title, image, author, summary, and price
- Static shopping cart page
- Navigation between pages
- Hover effects and basic UI styling

## Technologies
- HTML
- CSS
- CSS Grid
- Flexbox

## Purpose
This project demonstrates fundamental frontend development skills such as semantic HTML structure, responsive layouts, and clean CSS styling.
